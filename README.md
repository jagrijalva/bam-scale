# Acculturation_Political_Analysis

This repository provides transparent and reproducible access to data, analyses, and visualizations used to explore Latino political behavior via the Multidimensional Latino Acculturation Model (MLAM) using data from the 2006 Latino National Survey.

## Structure

- **Phase 1:** Data Preparation and setup scripts.
- **Phase 2:** Cluster analysis of Latino acculturation orientations.
- **Phase 3:** Political behavior analysis including Kruskal-Wallis, Dunn’s test, and Clifford’s Delta.
- **Presentations:** Conference slides and visuals.

## How to Use:
- Navigate into each folder for relevant scripts, data, and detailed PDF reports.
- All analyses are performed in R.
